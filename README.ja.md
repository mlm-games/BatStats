# BatStats

![バナー](fastlane/metadata/android/en-US/images/banner.svg)

Detailed Stats では、一部の統計情報が表示されない場合があります（デバイスによって異なります）。このアプリは大半のエッジケースには対応していませんが、全アプリの消費電力量（mAh）など、重要な統計についてはほぼ常に動作します。

## コントリビューション
Issue と PR を歓迎します。（提出前に、その問題が特定の OS に固有のものではないことをできる限り確認してください。）
