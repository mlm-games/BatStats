# BatStats

![横幅](fastlane/metadata/android/en-US/images/banner.svg)

Detailed Stats 可能不会显示某些统计数据（具体取决于设备）。本应用没有处理大多数边缘情况，但对于重要项目通常都能正常工作，例如所有应用的耗电量统计（单位为 mAh）。

## 参与贡献
